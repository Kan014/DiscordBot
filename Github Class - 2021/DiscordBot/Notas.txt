Antes de empezar no copien y peguen esto es para que se puedan relacionar mejor con su código.

Importante tener:

node V más reciente estable (No la v14)
discord.js v12 (Esta es la versión actual)

Si tienes un editor de código solo tienes que instalar node y discord.js

node => Debes ir a la página de node oficial
discord.js => npm i discord.js --save // Esto ejecutando la consola del mismo editor o el simbolo de sistema (cmd)

Si vas a usar un editor online, repl.it, glich, entre otros.

No hay problema con instalar el node, ya que generalmente este ya esta en las páginas "mencionadas"
